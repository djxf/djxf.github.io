考试重点

*常微分方程

选择题(20分)

  1. n阶微分方程解通解，特解。P17-18,P125定理6，P127定理7，P128定理1，P211定理7。
  2. 方程dy/dx=f(x,y)满足解的存在唯一性定理的矩形域R是().P77定理1，P85附注2。
  3. 方程M(x,y)dx + N(x,y)dy = 0的积分因子u=（）。P54（2.55）P58（例6）。
  4. 方程x’ = Ax的基解矩阵是()。P208，例1 P216.1
  5. 方程x‘’+ax‘+bx=f(t)有形如__的特解。P137-139，P145-148

填空题(20分)

  1. 已知dy/dx = f(x,y) ,y(x0) = y0.则||字母真难写 不打了。。。
  2. 方程dy/dx = f(x)g(y)的通解为___。P30-31，P42。
  3. 方程组x’=Ax满足初值条件 x(t0)=x0的第二次近视解为__。P197-201.
