阶梯式保险保价方案

近来参考了多家物流企业的保价方案，几乎无新意且大都相同。但是一家保险式的物流服务却让我眼前一亮。

来源：搜狐网

​

有别于传统保价服务，保价费低，赔付难赔付低使得公司和客户双方之间都难以满意，不知保险式保价能否成为解决之道。

在模拟数据下简单粗略分析方案如下。

某阶段快递丢失率：a%（1<a<2），损失保价价值分布。如价值在3000-5000时，保价费用b%。

价值在1-2000时保价费用为c%.

考虑在尽量满足客户的需求外，使公司整体保持不亏损。即满足：

​ 保价价值*（b%-a%）+ 运费 >= 0

由于运费的影响，可以看到当保价金额在1-2000时即使丢包，也不会发生亏损。

接下来我们考虑赔付标准，既然是保险赔付。我们简单考虑赔付100%，200%（2倍），300%（3倍）。

在只分析某一分段保价价值下的标准如1-2000（元）：

保价价值 X b% - 倍率X保价价值Xa% + 运费 >= 0

其中保价价值1-2000元，运费10-20，丢失包裹率1-2%，可使得在不亏损的情况下计算得保价费率b%。

保价费率 保价金额 赔付赔率
1.5% 价值*1.5% 1
3.5% 价值*3.5% 2
5.5% 价值*5.5% 3

再根据总体数据分布下可设计全局方案。由于数据有限且缺乏讨论。还需进一步分析。
